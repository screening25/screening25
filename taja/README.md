# Typin' Java - 자바 타자 연습 애플리케이션

이 프로젝트는 자바(Java) 코드와 핵심 개념을 타이핑하며 자연스럽게 학습할 수 있도록 돕는 웹 기반 타자 연습 애플리케이션입니다.

## ✨ 주요 기능

1.  **✌️ 두 가지 학습 모드**
    *   **Typing Practice**: 각 챕터의 핵심 예제 코드를 직접 타이핑하며 코드 구조와 문법에 익숙해질 수 있습니다.
    *   **Concept Study**: 변수, 객체지향, 예외처리 등 각 챕터의 상세한 개념 설명을 타이핑하며 이론을 복습하고 암기할 수 있습니다.

2.  **📚 챕터별 레슨 선택**
    *   자바의 기초부터 심화(네트워킹, 쓰레드 등)까지 챕터별로 레슨이 구성되어 있어 체계적인 학습이 가능합니다.
    *   'Select Lesson' 버튼을 통해 원하는 챕터와 세부 레슨을 쉽게 선택할 수 있습니다.

3.  **📊 실시간 타이핑 통계**
    *   타이핑하는 동안 분당 타수(WPM)와 정확도(Accuracy)가 실시간으로 계산되어 화면에 표시됩니다.
    *   연습 완료 후에는 최종 WPM, 정확도, 해당 레슨의 최고 WPM, 평균 WPM을 포함한 상세 결과를 확인할 수 있습니다.

4.  **💾 학습 상태 자동 저장 (DB 기능)**
    *   사용자의 마지막 학습 상태가 브라우저의 `localStorage`에 자동으로 저장됩니다.
    *   마지막으로 연습했던 **레슨, 학습 모드(코드/개념), 입력 내용**이 저장되어, 브라우저를 닫았다가 다시 열어도 이어서 학습할 수 있습니다.
    *   각 레슨과 모드별 타이핑 통계(최고 WPM, 평균 WPM 등)가 개별적으로 저장 및 관리됩니다.

5.  **⌨️ 편리한 입력 환경**
    *   Tab 키와 Enter 키 입력을 지원하여 실제 코딩 환경과 유사한 경험을 제공합니다.
    *   VS Code와 Dracula 테마에서 영감을 받은 깔끔하고 눈이 편안한 UI를 제공합니다.

## 🚀 실행 방법

1.  프로젝트 디렉토리에서 필요한 라이브러리를 설치합니다.
    ```bash
    npm install
    ```
2.  개발 서버를 실행합니다.
    ```bash
    npm run dev
    ```
3.  브라우저에서 `http://localhost:5173` 주소로 접속합니다.
